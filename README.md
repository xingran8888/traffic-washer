# 🌊 Traffic Washer - 流量冲刷器 v4.0

全国各省CDN覆盖 · 运营商识别 · 自动清理 · 直播源

## ✨ 功能特性

- 🌍 **全国15+省市CDN覆盖** — 清华/南大/阿里/腾讯/中科大/上交/哈工大/大工/重大等
- 📡 **运营商精确识别** — 电信/联通/移动/教育网，支持按运营商筛选
- 🧹 **自动清理无流量链接** — 每5分钟检测，剔除无效链接并随机补充
- 🕷️ **爬虫自动更新** — 每2小时自动爬取全国各省镜像站链接
- 📺 **IPTV直播源** — 央视/卫视/各省地方台 m3u8 直播流
- 🎬 **视频CDN** — B站/优酷/腾讯/爱奇艺 CDN 节点
- 🔒 **IPv6双栈** — 支持 IPv4/IPv6 双栈访问
- ⏰ **分时设置** — 支持3个时间段独立配置线程/流量/限速
- 📊 **实时统计** — 省份分布/运营商分布/实时速率/累计消耗

## 🚀 快速部署

### Docker 一键部署

```bash
# 克隆仓库
git clone https://github.com/xingran8888/traffic-washer.git
cd traffic-washer

# 构建镜像
docker build -t traffic-washer:latest .

# 运行容器
docker run -d --name traffic-washer --restart always -p 9999:9999 traffic-washer:latest
```

### 访问管理面板

打开浏览器访问: `http://你的服务器IP:9999`

## 📋 系统要求

- Python 3.12+
- Docker (推荐)
- 内存: 64MB+
- 网络: 需要公网访问

## 🔧 配置说明

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 线程数 | 20 | 同时下载线程数 |
| 每日流量限制 | 0 (无限) | GB为单位 |
| 限速 | 0 (不限) | MB/s为单位 |
| 省份筛选 | 所有 | 所有/同省/省外 |
| 运营商筛选 | 所有 | 电信/联通/移动/教育网 |
